Single Page Application Q&A

Q: Is this website a Single Page Application (SPA)?
A: Yes. By definition, SPA is a web site/application that fits in a single web page and all the necessary code (HTML, CSS, JS) is retrieved with a single page load.
Since this site meets these requirements - it is SPA.

Q: What is the main technological renewal of SPA?
A: the main technological renewal of SPA is the substantial reduce of server requests. The fact that the entire website is a single page, with no links to other pages, a few server requests are required in order to display the website. Hence the internet traffic reduces.

Q: What are the technical approaches of creating an SPA?
A: There are several approaches:

  • JavaScript Frameworks
    • AngularJS
    • Ember.js
    • Meteor.js
    • Aurelia
  • Ajax
  • Websockets
  • Server-sent events
  • Browser plugins
  • Data transport (XML, JSON and Ajax)
